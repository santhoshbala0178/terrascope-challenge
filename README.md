# terrascope-challenge

This project is an implementation of Memo Board

## Commands to install

- yarn install
- cd backend && yarn install (To install the requirements for Node Server)

## Commands to execute

- To Start the application - yarn start
- cd backend && yarn start (To start the Node Server)

## Tech Stack

- React
- Typescript
- Styled Components

## Major Components

- MemoPage
- Memo
- Add Button

## Features

- User can View, Edit, Delete or Add a memo
- Character limit of 140 and the notification to user if its nearing the limit
